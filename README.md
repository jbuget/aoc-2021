# aoc-2021
https://adventofcode.com/2021

## Installation

```bash
$ git clone git@github.com:jbuget/aoc-2021.git
$ cd aoc-2021
$ npm install
```

## Usage

```
$ npm start -- <day:number> <part:string>
```

Examples:

```bash
# ex: for part "one"
$ npm start -- 2 one

# ex: for part "two"
$ npm start -- 10 two 
```
